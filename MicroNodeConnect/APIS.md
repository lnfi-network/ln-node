### General Info APIs

#### initOwner
Request: 
```jsonc
{
    "method": "mn_initOwner",
    "params": {
        "name":"",
        "owner":"npub1234"
    }
}
```

Response:
```jsonc
{
    "result_type": "mn_initOwner",
    "result": {
        "nodeInfo": [
            
        ],
    },
}
```
